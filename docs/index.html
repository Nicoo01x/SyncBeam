Everything you need

Powerful features designed for seamless file sharing across your local network

Auto Discovery

Automatically finds other SyncBeam devices on your network using mDNS. No manual IP configuration needed.

Military-Grade Encryption

Noise Protocol XX handshake with AES-256-GCM encryption. Your data is protected with the same tech used by Signal.

Large File Support

Transfer files over 10GB with chunked streaming and automatic resume. No file size limits.

Clipboard Sync

Automatically sync your clipboard across devices. Copy on one PC, paste on another. Supports text, images, RTF and HTML.

Drag & Drop

Simply drag files to the app or drop them in the outbox folder. They'll be sent automatically to connected peers.

No Internet Required

Works entirely on your local network. No cloud servers, no internet connection needed, no data leaves your network.

Security first, always

SyncBeam was built with security as a core principle, not an afterthought. Every connection is encrypted end-to-end.

  • Ed25519 Identity Each device has a unique cryptographic identity
  • Noise Protocol XX Mutual authentication with identity hiding
  • AES-256-GCM Authenticated encryption for all data
  • Project Secret Only authorized peers can connect

How it works

Get started in seconds with these simple steps

1

Install SyncBeam

Download and install SyncBeam on two or more Windows PCs on the same network.

2

Share the Secret

Copy the project secret from Settings on one PC and paste it on the other. This ensures only your devices can connect.

3

Connect & Transfer

Devices appear automatically. Click Connect, then drag files or copy to clipboard. That's it!
